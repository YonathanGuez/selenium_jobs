# Selenium Jobs
Good practice for work with selenium in python

## Configuration:
Install selenium:
```
pip install selenium
```
Install Chromedriver:
```
https://sites.google.com/a/chromium.org/chromedriver/downloads
```
Take a good version of chrome and put this in :
```
C:/chromedriver/
```
